IntelliJ Ultimate 설치 및 학생인증

IntelliJ 버전

인텔리제이는 무료 버전인 Community 버전과 Ultimate버전이 있습니다. Community 버전으로도 Java 개발을할 수 있지만 eclipse로 Maven프로젝트를 해보신 분이라면 아시겠지만 설정이 매우 복잡해집니다. 하지만 Ultimate 버전을 사용하게되면 백엔드 작업을 위한 Spring, Java EE 뿐만 아니라 DB도구, 쿠버네티스관련 기능 그리고 JS개발기능까지 지원합니다. 따라서 웹 프로젝트를 하신다면 Ultimate버전을 사용하는것이 편리합니다.

학생인증

  1. 아래 링크를 통해 Jetbrains 학생인증 페이지에 접속하여 "Apply now" 클릭

    https://www.jetbrains.com/community/education/#students

  2. 학교 이메일 정보 입력 및 신청
  3. 입력한 학교 이메일에 온 인증메일에서 "Confirm Request" 클릭
  4. 링크를 누르면 JetBrains페이지로 넘어가 "I Accept"를 클릭
  5. 회원 가입이 되어있지 않은 경우 회원 가입 진행
  6. 회원 가입 완료 후 가입한 이메일에 메일을 확인하여 "Confirm your account"클릭
  7. 나머지 정보를 입력하여 회원 가입 완료
  8. JetBrains 로그인하여 라이센스 확인

IntelliJ Ultimate 설치

  1. 아래의 링크로 가서 Ultimate 버전을 다운받습니다.

    https://www.jetbrains.com/idea/download/#section=windows

  2. 최초 인증 계정 정보 입력으로 인증 완료

IntelliJ IDEA 단축키 15가지

단축키 목록

단축키 설명
Alt + Enter 현재 커서에서 발견된 오류, 경고, 제안에 대한 수정안을 제안
F2 다음 오류, 경고, 제안으로 점프
Alt + 1 Project창을 열어주고 커서를 Project창으로 이동
Esc 에디터로 커서를 이동
Ctrl + E 최근 열어본 파일을 보여줌
Ctrl + B 심볼의 선언 위치로 이동
Alt + F7 해당 심볼의 사용처를 보여줌
Ctrl + Ctrl 자신이 원하는 것을 실행
Ctrl + W/ Shift + Ctrl + W 해당 코드를 감싸는 코드를 선택해준다(Shift를 누르면 반대로)
Ctrl + / 해당 줄을 주석으로 만듦(여러 줄을 선택했을 경우 블록주석)
Shift + Ctrl + Enter 자동 완성
Ctrl + Alt + L 코드이 스타일을 맞춰준다(띄어쓰기, 화살함수 사용 등)
Shift + Ctrl + Alt + T 리팩토링(심볼을 하나하나 바꾸지 않고 한번에 바꾸거나 파라미터를 바꾸는 등)
Shift + Ctrl + A Actions 검색
Shift + Shift 전체 검색, 모든 항목에 대한 검색
